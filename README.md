
  # Modern Website

  This is a code bundle for Modern Website. The original project is available at https://www.figma.com/design/sBcl5w9iifZwUK5nXzSuOP/Modern-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start both the Vite frontend and the API server.

  If you only need the API, run `npm run server:dev`.
